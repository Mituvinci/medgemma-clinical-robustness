  5. What Happens When You Click "Analyze Case"

  The Workflow Executes:

  User clicks button
      ↓
  Gradio calls process_case_async()
      ↓
  Creates ClinicalCase object
      ↓
  Calls workflow.run_async(case)
      ↓
  ┌─────────────────────────────────────────┐
  │ HYBRID WORKFLOW EXECUTES                │
  ├─────────────────────────────────────────┤
  │ Step 1: Root Coordinator (ADK/Gemini)   │
  │   → Delegates to TriageAgent            │
  │                                         │
  │ Step 2: TriageAgent (ADK/Gemini)        │
  │   → Calls analyze_case_completeness()  │
  │   → Calls medgemma_triage_analysis() ← MedGemma! │
  │                                         │
  │ Step 3: ResearchAgent (ADK/Gemini)      │
  │   → Calls retrieve_clinical_guidelines()│
  │   → Queries ChromaDB (1,492 chunks)    │
  │   → Calls medgemma_guideline_synthesis() ← MedGemma! │
  │                                         │
  │ Step 4: DiagnosticAgent (ADK/Gemini)    │
  │   → Calls medgemma_clinical_diagnosis() ← MedGemma! │
  │   → MedGemma generates SOAP note        │
  └─────────────────────────────────────────┘
      ↓
  Response returns to UI
      ↓
  UI parses and displays results

  Duration: 30-60 seconds (MedGemma API calls take time)
